# SPA Приложение для управления товарами

Данный проект представляет собой одностраничное приложение (SPA), реализующее полный цикл CRUD-операций с сущностью «Товар».

## Используемые технологии
* React
* React Router DOM (для роутинга)
* Axios (для запросов к API)
* JSON-Server (для имитации backend-сервера)

## Инструкция по запуску локально

1. Установите все необходимые зависимости:
   npm install

2. Запустите локальный mock-сервер базы данных (выполнять в первом терминале):
   json-server --watch db.json --port 5000

3. Запустите сервер разработки React (выполнять во втором терминале):
   npm start

После сборки приложение будет доступно в браузере по адресу http://localhost:3000
